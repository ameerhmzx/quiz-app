## Quiz App

A simple application for teachers to conduct quick quiz.

### Run Locally

> create `.env` file using `.env.example` as a sample
> 
> Make sure to run a postgres database before proceeding

```bash
npx prisma migrate deploy
npx prisma generate 
npm run dev
```
